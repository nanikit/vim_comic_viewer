# vim comic viewer

tampermonkey에서 쓰려 만든 만화 뷰어 라이브러리입니다.

조작 단축키는

- <kbd>i</kbd>: 전체 화면 전환

- <kbd>j</kbd>, <kbd>↓</kbd>: 다음 페이지

- <kbd>k</kbd>, <kbd>↑</kbd>: 이전 페이지

- <kbd>;</kbd>: 전체 이미지 다운로드

- <kbd>/</kbd>, <kbd>?</kbd>: 페이지 짝 맞춤 조절

- <kbd>'</kbd>: 로딩되지 않은 이미지 새로고침

## 개발자 전용

최신 사용 예제는 [여기](https://github.com/nanikit/comic_sources) 있습니다.
